HYBRIDSYNCHAADL: Modeling and Formal Analysis of Virtually Synchronous CPSs in AADL

This web page contains resources for our paper “HYBRIDSYNCHAADL: Modeling and Formal Analysis of Virtually Synchronous CPSs in AADL” by J. Lee, S. Kim, K. Bae, P. C. Ölveczky, and M. Kang.

HybridSynchAADL Tool

HybridSynchAADL is an OSATE plug-in to model Hybrid Synchronous AADL models in the OSATE environment. Our tool requires the following tools:

  • Java 1.8: For running OSATE, Java 8 or upper version is required
  • Mause with SMT solving: HybridSynchAADL uses Maude combined with SMT solving to symbolically enoce all possible continuous behaviors
After install all the tools above, download HybridSynchAADL tool in this link. After extracting the zip file, click "osate" to execute HybridSynchAADL tool. You can see how to anlayze HybridSynchAADL model using the tool in tutorial.pdf

Benchmarks

There are six benchmarks used in the HybridSynchAADL tool. Each benchmarks are as follows:

  • Drone_Single_Integrator: rendezvous and formation control of drones with single-integrator dynamics
  • Drone with double-integrator: rendezvous and formation control of drones with single-integrator dynamics
  • Thermostat: networked thermostat system
  • WaterTank: watertank system with pump
  • FourDronesExample: rendezvous control of four one-dimensional drones
  • FourDronesSystem_Two_Dim: rendezvous control of four two-dimensional drones
For the convenience, we also provide the zip file that contains all benchmarks.
